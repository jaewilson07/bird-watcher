# Diagrams

Visual aids for the tutorial.

- `00-master.png` / `.excalidraw` — the kid-friendly big picture (5 boxes)
- `00-technical.png` / `.excalidraw` — the same system, more detail (for the curious)
- `01-step.png` through `10-step.png` — one per issue, highlighting what THIS issue builds

Each `NN-step.excalidraw` is editable in https://excalidraw.com — open it, modify, save, regenerate the PNG.

The diagrams are referenced from:
- `README.md` — the master diagram is the cover image
- each issue body on GitHub — embeds `NN-step.png` so the student sees what they're building
- the first cell of each tutorial notebook — same image, same context
